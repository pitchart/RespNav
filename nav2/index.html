Mon menu :hover

  • Salade
  • Tomate
  • Oignons
  • Choucroute
  • Bretzel
  • Picon bière

Avantages :

  • compatibilité maximale
  • simple à mettre en place, pas de bidouilles ni de code HTML inutile, à part un onclick=”” pour certains navigateurs (Firefox mobile, Opera mini)
  • pas de requête et traitement JS nécessaire

Inconvénients :

  • ne correspond pas au même événement sur desktop (survol) et sur mobile (touch)
  • on ne peut pas refermer simplement le menu en recliquant dessus, il faut cliquer sur un lien ou hors du menu
  • ciblage limité en CSS (on ne peut avoir d’action que sur un frère ou un enfant)

Compatibilité :

  • OK sur ordinateurs de bureau
  • OK sur mobile : iOS 5+, Android 2.1+, Opera Mobile, Blackberry 6+, IE 9+, Firefox, Chrome
  • Anomalie : Opera Mini affiche le menu déployé et ne permet pas de le replier
  • OK avec le lecteur d'écran VoiceOver
  • Compatibilité mobile de :hover
